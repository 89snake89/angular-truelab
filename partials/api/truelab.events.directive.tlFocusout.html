tlFocusout
directive in module truelab.events

Description

The tlFocusout directive allows you to specify custom behavior on element focus out.

Usage

as attribute
<ANY tl-focusout="{expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
tlFocusoutexpression

Expression to evaluate upon focus out. (Event object is available as $event)
