tlFirstUpper
filter in module truelab.strings.filters

Description

First letter to uppercase

Usage

In HTML Template Binding

{{ tlFirstUpper_expression | tlFirstUpper:each }}

In JavaScript

$filter('tlFirstUpper')(text, each)

Parameters

ParamTypeDetails
textstring
  • text to capitalize
eachboolean
  • if true capitalize first letter in each word

(default: false)
