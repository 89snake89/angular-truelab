tlLoadImage
directive in module truelab.loadImage

Description

Append image only when is loaded, add css state classes on loading and on error to control visualization

Dependencies

  • $tlLoadImage
  • $tlLoadImageOptions

Usage

as attribute
<ANY tl-load-image="{string}"
     tl-load-image-options="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
tlLoadImagestring
  • the image src
tlLoadImageOptionsstring
  • an object for directive options
