tlTruncate
filter in module truelab.strings

Description

Truncate a string if it's too long and add a suffix at the end

Dependencies

  • $tlStringUtils

Usage

In HTML Template Binding

{{ tlTruncate_expression | tlTruncate:length:end }}

In JavaScript

$filter('tlTruncate')(text, length, end)

Parameters

ParamTypeDetails
textstring
  • string to truncate
lengthint
  • the desired output length

(default: 10)

endstring
  • string to append at the end

(default: '...')
