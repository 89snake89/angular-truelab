tlClickout
directive in module truelab.events

Description

The tlClickout directive allows you to specify custom behavior on element click out.

This directive requires jQuery library.

Usage

as attribute
<ANY tl-clickout="{expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
tlClickoutexpression

Expression to evaluate upon click out. (Event object is available as $event)
