$tlLoadImageOptions
service in module truelab.loadImage

Description

tlLoadImage directive default options, override this value to customize all tlLoadImage directives.

Default object
{
  delay : 0,
  css : {
    base : 'tl-load-image',
    loading : 'loading',
    error : 'error'
  }
}

Properties

  • [delay=0]

    • delay applied when loads images

  • [css.base='tl-load-image'']

    • base css class

  • [css.loading='loading']

    • loading css class

  • [css.error='error']

    • error css class
