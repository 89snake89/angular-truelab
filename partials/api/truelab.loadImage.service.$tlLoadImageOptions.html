$tlLoadImageOptions
service in module truelab.loadImage

Description

tlLoadImage directive default options, override this value to customize all tlLoadImage directives.

Default object
{
  delay : 0,
  css : {
    base : 'tl-load-image',
    loading : 'loading',
    error : 'error'
  }
}

Properties

  • delay

    • delay applied when loads images ( default : 0 )

  • css.base

    • base css class (default : 'tl-load-image')

  • css.loading

    • loading css class (default : 'loading')

  • css.error

    • error css class (default : 'error')
